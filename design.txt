Primera Version:
Realizaremos un diseño orientado a objetos y utilizaremos el patron de diseño Modelo-Vista-Controlador (MVC)
Para la parte del modelo tendremos los objetos Miner, DataBase, Rola, RequestProcessor.

Miner se encargara del minado para la base de datos de nuestro programa. Recorrera el arbol del directorio que el usuario proporcione en busca de archivos mp3 de los cuales, guardara su ruta en una estructura que tendra como propiedad. Una vez localizadas todas las rutas de los archivos mp3 en el directorio se encargara de obtener las etiquetas ID3v2 indicadas de cada archivo. Para esto creara una nueva Rola con todos los metadatos obtenidos de cada archivo para posteriormente tratar de guardarlas en la base de datos.

Rola solo tendra como propiedades el titulo, artista, album, año, genero, track, id y su ruta en el directorio.

DataBase en encargara de todo lo relacionado a la base de datos, crearla, agregar, borrar y buscar registros, actualizar las tablas, etc.

RequestProcessor tendra comunicacion directa con la base de datos y (dependiendo del tipo de consulta solicitada) regresara la informacion para ser mostrada al usuario.
